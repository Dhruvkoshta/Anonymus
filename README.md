## üöÄ Getting Started

### Cloning the Repository

To clone the repository locally, use the following commands:

```bash
git clone
cd anonymus
```

### Installation
```bash
bun install
```
### Configuration
Create a `.env` file in the root folder of your project. Update it following the convention of the `.env.example` file. 
Here's an example:
```
DATABASE_URL=
JWT_SECRET=
GEMINI_API_KEY=
RESEND_API_KEY=
RESEND_FROM_EMAIL=
```

### Running the Project
```bash
bun run dev
```
